# ElyDoc Design Principles

## Overall Design Direction

ElyDoc should feel like modern Irish medical minimalism.

The site should not feel like:

- a generic GP website
- a cheap online doctor service
- a wellness influencer brand
- a Silicon Valley biohacker brand
- a corporate hospital website
- a pharmaceutical sales funnel

It should feel:

- calm
- premium
- modern
- discreet
- clinical
- warm
- trustworthy
- elegant
- editorial
- highly readable

Design inspiration:

- Apple
- Manual
- Numan
- Levels Health
- Maven Clinic
- premium healthcare SaaS
- Scandinavian wellness design
- high-end editorial health publications

## Colour Palette

Primary colour:

Deep Forest Green  
#1F3A34

Use for:

- primary buttons
- key accents
- selected backgrounds
- trust sections
- navigation highlights

Primary background:

Warm Ivory  
#F7F5F1

Use for:

- main site background
- calm premium sections

Primary text:

Graphite  
#1C1C1C

Use for:

- headings
- body text
- main UI text

Secondary accent:

Muted Sage  
#A8B5A2

Use for:

- badges
- subtle backgrounds
- decorative details
- soft cards

Neutral border/card tone:

Warm Stone  
#D8D3CB

Use for:

- borders
- dividers
- card outlines
- subtle section contrast

Optional premium accent:

Muted Champagne  
#C2A878

Use very sparingly.

Do not overuse gold.

Avoid:

- bright medical blue
- neon green
- harsh red except for urgent safety warnings
- excessive gradients
- overly colourful sections

## Typography

Recommended:

Headings:
- Geist
- or Inter Tight

Body:
- Inter

Optional editorial accent:
- Cormorant Garamond, used sparingly only for subtle premium editorial emphasis

Typography should be:

- clean
- spacious
- modern
- readable
- not overly bold
- not playful
- not corporate-heavy

Headings should use:

- generous line height
- medium weight
- strong hierarchy
- restrained wording

Body text should use:

- comfortable line height
- readable font size
- constrained text width
- good contrast

## Layout

Use:

- generous whitespace
- large section padding
- calm vertical rhythm
- constrained content widths
- clear hierarchy
- mobile-first layouts
- simple grids
- elegant cards

Avoid:

- cramped sections
- dense walls of text
- too many cards per row
- overloaded hero sections
- excessive visual noise

## UI Components

Buttons:

- rounded-xl or rounded-2xl
- calm hover transitions
- primary button: deep forest background with ivory text
- secondary button: outline or ghost style
- no aggressive gradients
- no loud shadows

Cards:

- subtle border
- soft shadow
- rounded-2xl
- generous padding
- minimal internal clutter

Badges:

- small
- elegant
- muted colours
- useful for trust signals, service categories and clinical safety notes

## Animation

Use Framer Motion only subtly.

Acceptable:

- fade in
- slight upward motion
- soft hover states
- gentle accordion transitions
- subtle reveal on scroll

Avoid:

- bouncing
- spinning
- excessive parallax
- flashy hero animations
- distracting motion

## Imagery

Avoid:

- cheesy stock doctor photos
- fake smiling medical teams
- obvious AI-generated faces
- overly clinical hospital imagery
- generic handshake business images

Prefer:

- abstract healthcare visuals
- calm interiors
- editorial lifestyle photography
- soft natural light
- minimalist illustrations
- organic forms
- subtle Irish landscape/heritage references if tasteful
- premium textures
- clean iconography

## Logo/Brand Symbol

The ElyDoc logo may reference Ely and O'Carroll heritage through circles/arcs, but the website should not over-explain this.

The brand mark should feel:

- timeless
- restrained
- elegant
- symbolic
- premium
- not overly medical

## Design Rule

Every page should pass this test:

"Does this feel like a doctor-led premium healthcare service that I would trust with my personal health information?"

If not, simplify.
